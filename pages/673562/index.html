# 数据集

VRP问题有一些标准数据集，这些数据集大部分都是有已知的最优解的，我们可以使用这些标准数据集来验证检验自己算法的健壮性，以下是我整理的部分数据集的网站，推荐指数从0到5不等，0代表我还没有使用过。

网站 描述 推荐指数
OR-Library (opens new window) 这个网站涵盖了大部分运筹优化问题的数据集，其中就包括了VRP问题，涵盖单车场、多车场、同时送取货、时间窗等多种类型的数据集。 0
Solomen benchmark (opens new window) 这个网站主要是PDPTW和VRPTW的数据集，客户点数量从0至1000个不等。 0
CVRPLIB (opens new window) 顾名思义，包含的是CVRP问题的相关数据集。 0
DIMIACS (opens new window) 离散数学和理论计算机科学中心，一个不明觉厉的网站，包含了八种VRP问题介绍及其数据集 5
